# URP Geometry Shader Example
![](/Images/animated-extrusion.gif)

Some examples of how can you use geometry shader in Unity URP.

## Requirement
- Unity 2019.3.1f1
- Universal Rendering Pipeline 7.3.1
- Geometry shader support, obviously
